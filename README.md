## Building a Ground-Up Neural Network Framework With Convolutional Support

## Intro paragraph about what it does and how its used for SEO
This project implements a custom neural network framework built from the ground up, without relying on existing deep learning libraries. The framework supports fully connected and convultional models, manual forward and backward propagation, and CPU/GPU exection via NumPy and CuPy.
